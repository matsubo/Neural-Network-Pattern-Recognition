Usage
---

- Windows

```
 git clone git@github.com:matsubo/Neural-Network-Pattern-Recognition.git
 cd Neural-Network-Pattern-Recognition
 start.bat
```


- UNIX (Mac OS X, Linux, X Window System)

```
 git clone git@github.com:matsubo/Neural-Network-Pattern-Recognition.git
 cd Neural-Network-Pattern-Recognition
 ./start.sh
```
